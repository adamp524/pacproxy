# Pacproxy

Pacproxy provides http/https proxy routed with proxy.pac.

## Installation

Add this line to your application's Gemfile:

    gem 'pacproxy'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pacproxy

## Usage

    $ bundle exec ruby bin/pacproxy -P proxy.pac -p 3128
